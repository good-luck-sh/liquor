🍶 전통주情(정)이란
---
코로나 19확산으로 흥하고 있는 혼술시장에 맞춰 </br>
혼술시장을 공략한 전통주를 구매할 수 있는 구매페이지를 만들어 </br>
사람들과 의견을 공유하고,  </br>
코로나 19가 사라진다면 사람들이 전통주 만드는 지역의 향토음식을 확인하는  </br>
전통주와 정에 대한 웹 페이지를 구현하였다. </br>
